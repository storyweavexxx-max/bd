## ✨ ফিচারসমূহ

- 🎨 **সুন্দর Dual Theme** — ডার্ক ও লাইট মোড (সুন্দর টগল সুইচ সহ)
- 🖼️ **অটোমেটিক হিরো স্লাইডার** (5টি হাই-কোয়ালিটি স্লাইডার ইমেজ)
- 🔍 **পাওয়ারফুল সার্চ** — গল্পের নাম ও ট্যাগ দিয়ে
- 📂 **ক্যাটাগরি ফিল্টার** — বউ, শালী, ভাবি, পরকীয়া, অজাচার, মা-ছেলে ইত্যাদি
- ❤️ **গল্প সেভ** (Favorites) সুবিধা
- 💬 **মন্তব্য সিস্টেম** প্রত্যেক গল্পে
- 👑 **অ্যাডমিন প্যানেল** (`#admin` দিয়ে অ্যাক্সেস)
- 📱 **Fully Responsive** — মোবাইল, ট্যাবলেট ও ডেস্কটপের জন্য অপটিমাইজড
- ⚡ **Fast & Lightweight** — Vanilla JavaScript + Tailwind CSS

## 🚀 কীভাবে লোকালে রান করবেন

```bash
# ১. রিপোজিটরি ক্লোন করুন
git clone https://github.com/storyweavexxx-max/bd.git

# ২. ফোল্ডারে প্রবেশ করুন
cd bd

# ৩. index.html ফাইলটি Live Server দিয়ে খুলুন
# অথবা সরাসরি ব্রাউজারে খুলুন
```

### Firebase Setup (ঐচ্ছিক)
- Firebase Console-এ প্রজেক্ট তৈরি করুন
- Firestore Database চালু করুন
- `index.html` ফাইলে `firebaseConfig` আপডেট করুন
- Firestore Rules সেট করুন (ডেভেলপমেন্টের জন্য সিম্পল রুলস ব্যবহার করুন)

## 🛠️ ব্যবহৃত প্রযুক্তি

- **Frontend**: HTML5, Tailwind CSS v4, Vanilla JavaScript
- **Backend**: Firebase Firestore
- **Icons**: Font Awesome 6
- **Font**: Hind Siliguri (বাংলা সাপোর্ট)

## 👑 অ্যাডমিন অ্যাক্সেস

- অ্যাডমিন প্যানেল খুলতে: **`yourdomain.com/#admin`**
- ডিফল্ট পাসওয়ার্ড: **`admin123`**
